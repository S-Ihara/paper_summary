# Paper Summary
LLMに論文を要約させるやつ  
無料で使いやすいgeminiのapiを使っています  

### 使い方
- 設定ファイル
    - `mv config_template.py config.py`
    - してconfigの中にAPIのKEYを設定する

- 実行
    - ./papers/pdf/* に要約したいpdfのファイルを置いておく
    - main.pyで要約ファイルが./paeprs/markdown/* に作成される
